post_repository.dart

File-level Dartdoc: Singleton repository that manages local posts and map pins, persists them to the application's documents directory, and coordinates uploads to the remote PostsApi when available. It also exposes a ValueNotifier for UI actions (for example requesting a tab switch) and helpers to read/write a stored country code via SharedPreferences.
